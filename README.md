# 🤖 Тілтабыс Боты

Ұйымдағы қақтығыстардың алдын алуға арналған Telegram боты.

## Мүмкіндіктер
- 📋 Конфликтке бейімділік тесті (5 сұрақ)
- 🧘 Стресті төмендету әдістері
- ✍️ Мәтіндік терапия (ойды жазу)
- 📞 Психологиялық қолдау орталықтары туралы ақпарат

---

## Railway-де орналастыру (тегін)

### 1. Файлдарды GitHub-қа жүктеу
1. [github.com](https://github.com) сайтына кіріп, жаңа репозиторий жасаңыз
2. Осы папкадағы барлық файлды жүктеңіз:
   - `bot.py`
   - `requirements.txt`
   - `Procfile`

### 2. Railway-де жобаны жасау
1. [railway.app](https://railway.app) сайтына кіріңіз (GitHub арқылы тіркеліңіз)
2. **"New Project"** → **"Deploy from GitHub repo"** → репозиторийіңізді таңдаңыз
3. Жоба автоматты жасалады

### 3. Бот токенін қосу
1. Railway жобасында **"Variables"** бөліміне кіріңіз
2. **"Add Variable"** батырмасын басыңыз
3. Мынаны қосыңыз:
   ```
   BOT_TOKEN = (сіздің токеніңіз)
   ```
4. **"Deploy"** батырмасын басыңыз

### 4. Дайын! ✅
Бот іске қосылады. Telegram-да `/start` деп жазып тексеріңіз.

---

## Render-де орналастыру (баламалы)

1. [render.com](https://render.com) сайтына кіріңіз
2. **"New"** → **"Web Service"** → GitHub репозиторийін жалғаңыз
3. Мынаны орнатыңыз:
   - **Environment:** Python
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python bot.py`
4. **Environment Variables** бөлімінде `BOT_TOKEN` қосыңыз
5. **"Create Web Service"** басыңыз

---

## Жергілікті тестілеу

```bash
# Тәуелділіктерді орнату
pip install -r requirements.txt

# .env.example файлын көшіріп, токенді қосыңыз
cp .env.example .env
# .env файлын ашып, токенді жазыңыз

# Ботты іске қосу
python bot.py
```
